# IBM watsonx

Explore the IBM watsonx platform through product overviews, real-world use cases, live demos, and downloadable assets — designed to accelerate understanding and adoption.

## Platform Overview

IBM watsonx is an integrated AI and data platform built on a hybrid cloud foundation. It brings together tools for building, deploying, and governing AI across the enterprise.

### Core Products

- **watsonx.ai** — AI development studio with foundation models (Granite, Llama, Mistral), prompt engineering, RAG, fine-tuning, and agent lifecycle management.
- **watsonx.data** — Open data lakehouse with fit-for-purpose query engines (Presto, Spark), vector search, and built-in data integration, governance, quality, and lineage.
- **watsonx Orchestrate** — Agent builder for creating AI assistants and multi-agent workflows using no-code, low-code, and pro-code tools.
- **watsonx.governance** — AI governance with model inventory, risk assessments, drift and bias monitoring, content safety guardrails, and regulatory compliance (EU AI Act, ISO 42001).

### Platform Layers

- **Automation** — AI-powered automation for app modernization, hybrid integration, infrastructure provisioning, identity governance, FinOps, and asset lifecycle management.
- **Hybrid Cloud** — Red Hat OpenShift, RHEL, and Ansible running on IBM Z, Power, and cloud infrastructure with GPU acceleration for AI workloads.
